# photoProcessing

## Development

### Installing Dependencies
From within the root directory
```sh
npm install
nodemon
``

### API Key Set-Up:
#### Create api-key.js
Using 
```sh
Copy api-key-sample.js and rename to api-key.js
```
#### S3 access keys
* Set-up S3 account
* Obtain access keys, bucket and region
* Update api-key.js

#### Clarifai
* Obtain access keys [Clarifai](https://developer.clarifai.com/signup/)
* Update api-key.js
